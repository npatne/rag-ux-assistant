*   **Poor Control Discrimination:** The chains were too similar.
*   **Weak Memory Encoding:** Users had trouble remembering which chain did what.

This reminds me of a simpler problem, but a similar idea:

*   **Weak Memory Encoding:** If you just *remember* which potato controls what, it is difficult. You need to be able to clearly label each potato.
*   **Poor Feedback:** If you can't tell the fan speed, it's like not seeing how quickly the potatoes are spinning. You might need an indicator to show the speed.

Query 13: Why do toast land butter side down?
Answer: Based on the study, participants found the pull-chain ceiling fans difficult to use. The main issues they encountered were:

*   **Trouble with control discrimination:** The chains were too similar, making it hard to tell which one controlled the speed and which controlled the light.
*   **Weak memory encoding:** Users had to remember which chain did what and which speed setting they were on. This was difficult.
*   **Poor feedback:** There wasn't a good way to tell what speed the fan was set to.

This resulted in frustration and accidental activation of the wrong controls.
